# Task 8 – Simple Sales Dashboard (Power BI)

##  Objective
Create a simple and interactive sales dashboard showing performance by month, region, and category using Power BI.

---

##  Dataset
Dataset Used: **Superstore_Sales.csv**

Columns include:
- Order Date  
- Region  
- Category  
- Sales  
- Profit  

---

##   Tools Used
- Power BI Desktop

---

##  Dashboard Visuals

###  1. Line Chart – Monthly Sales Trend
- X-axis: Month-Year  
- Y-axis: Sales  
- Shows change in sales over time

###  2. Bar Chart – Sales by Region
- Displays revenue from each region  
- Sorted highest to lowest

###  3. Donut Chart – Sales by Category
- Shows category contribution to total sales

###  4. Slicer
- Region filter for interactivity

---

##  Key Insights
1. **West region recorded the highest sales.**  
2. **Technology category contributed the largest share of sales.**  
3. **Sales peak during November and December.**  
4. **Central region has the lowest revenue.**
