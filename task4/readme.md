# Task 4 – Dashboard Design (Power BI)

##  Objective
The goal of this task is to create a simple and interactive dashboard in Power BI using the Superstore dataset.  
The dashboard presents key business insights using KPI cards, charts, and slicers.

---

##  Dataset
**Dataset Used:** Sample Superstore (Kaggle)

Columns include:
- Order Date  
- Sales  
- Profit  
- Order ID  
- Category  
- Region  
- State  
- Customer, Product, Ship Mode etc.

---

##  Tools Used
- Power BI Desktop  
- Sample Superstore Dataset  
- GitHub (for submission)

---

##  Dashboard Contents
The dashboard contains the following visuals:

###  1. KPI Cards
- **Total Sales**  
- **Total Profit**  
- **Total Orders (Count of Order ID)**  
- **Average Sales**

###  2. Monthly Sales Trend (Line Chart)
- X-axis: Order Date  
- Y-axis: Sales  
- Shows monthly sales over time  

###  3. Profit by Category (Bar Chart)
- Category vs Profit  
- Compare profitability across Furniture, Office Supplies, and Technology  

###  4. Sales by Region (Donut Chart)
- Shows % contribution of West, East, Central, South  
- Helps identify high-performing regions  

###  5. Slicers for Interactivity
- **Order Date (Year)**  
- **Category**  
- **Region**  

Users can filter the dashboard using the slicers to view insights for specific segments.
